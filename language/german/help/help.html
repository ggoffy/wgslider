Hilfe: wgSlider

wgSlider ist ein Modul für das XOOPS CMS, mit dem Administratoren Bilder-Slideshows einfach erstellen und verwalten können. Bilder werden in Kategorien organisiert und als Slider über Blöcke oder Smarty-Variablen angezeigt.

Funktionen

  • Kategorieverwaltung
  • Bild-Upload und Verwaltung
  • Mehrere Slideshow-Typen
  • Unterstützung für Bootstrap-Slideshows
  • Anzeige als Block oder Smarty-Variable
  • Integrierter Bildeditor
  • Sortierung der Bilder nach Gewicht
  • Statuskontrolle (online/offline)
  • Unterstützung für Modulkopien

Installation

  1. Modulordner kopieren nach
    /modules/wgslider
  2. XOOPS-Administration öffnen
  3. Gehe zu
    System → Module
  4. wgSlider installieren

Modulstruktur

Das Modul basiert auf drei Hauptelementen:

  • Kategorien
  • Bilder
  • Slideshows

Kategorien

Kategorien organisieren Bilder und legen fest, wie sie angezeigt werden. Jede Kategorie kann eine andere Slideshow verwenden.

  • Name
  • Anzeigetyp
  • Maximale Anzahl an Bildern
  • Bildbreite und -höhe
  • Status
Tipp: Erstellen Sie mehrere Kategorien, um unterschiedliche Slider auf verschiedenen Seiten anzuzeigen.

Bilder

Bilder werden hochgeladen und den Kategorien zugeordnet. Jedes Bild kann zusätzliche Informationen enthalten.

  • Name
  • Tooltip-Text
  • Gewicht (Anzeigereihenfolge)
  • Status
  • Einreicher

Slideshows

Slideshows legen fest, wie Bilder präsentiert werden. Verschiedene Templates können verwendet werden.

Beispiele:
  • Standard-Slideshow
  • Bootstrap 3 Carousel
  • Bootstrap 5 Carousel

Slideshows anzeigen

Mit Blöcken

Slideshows können als XOOPS-Blöcke angezeigt werden.

Schritte:
  1. Gehe zu
    System → Blöcke
  2. Block wgSlider Slideshow aktivieren
  3. Kategorie auswählen
  4. Blockposition wählen
Mit Smarty-Variablen

Wenn eine Kategorie mit einem Smarty-Key konfiguriert ist, kann sie direkt in einer Vorlage verwendet werden.

Beispiel:
<{$wgslider_meinekategorie}>

Bildeditor

wgSlider enthält einen integrierten Bildeditor.

Verfügbare Werkzeuge:
  • Bild zuschneiden
  • Bild drehen
  • Bild spiegeln
  • Ein-/Auszoomen
  • Seitenverhältnis kontrollieren

Moduleinstellungen

Folgende Einstellungen können in den Moduleinstellungen konfiguriert werden:

  • Maximale Uploadgröße
  • Erlaubte MIME-Typen
  • Maximale Bildbreite
  • Maximale Bildhöhe
  • Admin-Pagination
  • Soziale Lesezeichen

Tipps

  • Verwenden Sie Bilder mit identischer Größe für ein optimales Slider-Layout
  • Verwenden Sie Gewichtsangaben, um die Bildreihenfolge zu steuern
  • Nutzen Sie Bootstrap-Themes für beste Kompatibilität
  • Fügen Sie Tooltips für besseres SEO hinzu
