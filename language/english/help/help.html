Help: wgSlider

wgSlider is a module for the XOOPS CMS that allows administrators to create and manage image slideshows easily. Images are organized into categories and displayed as sliders using blocks or template variables.

Features

  • Category management
  • Image upload and management
  • Multiple slideshow types
  • Bootstrap slideshow support
  • Display as block or Smarty variable
  • Integrated image editor
  • Image weight sorting
  • Status control (online/offline)
  • Module cloning support

Installation

  1. Copy the module folder into
    /modules/wgslider
  2. Open XOOPS administration
  3. Go to
    System → Modules
  4. Install wgSlider

Module Structure

The module is based on three main elements:

  • Categories
  • Images
  • Slideshows

Categories

Categories organize images and define how they will be displayed. Each category can use a different slideshow.

  • Name
  • Display type
  • Maximum number of images
  • Image width and height
  • Status
Tip: Create multiple categories to display different sliders on different pages.

Images

Images are uploaded and assigned to categories. Each image can contain additional information.

  • Name
  • Tooltip text
  • Weight (display order)
  • Status
  • Submitter

Slideshows

Slideshows define how images are presented. Different templates can be used.

Examples:
  • Default slideshow
  • Bootstrap 3 Carousel
  • Bootstrap 5 Carousel

Displaying Slideshows

Using Blocks

Slideshows can be displayed as XOOPS blocks.

Steps:
  1. Go to
    System → Blocks
  2. Enable block wgSlider Slideshow
  3. Select category
  4. Choose block position
Using Smarty Variables

If a category is configured with a Smarty key, you can use it directly inside a template.

Example:
<{$wgslider_mycategory}>

Image Editor

wgSlider includes an integrated image editor.

Available tools:
  • Crop image
  • Rotate image
  • Flip image
  • Zoom in/out
  • Aspect ratio control

Module Preferences

The following settings can be configured in the module preferences:

  • Maximum upload size
  • Allowed MIME types
  • Maximum image width
  • Maximum image height
  • Admin pagination
  • Social bookmarks

Tips

  • Use images with identical sizes for best slider layout
  • Use weight values to control image order
  • Use Bootstrap themes for best compatibility
  • Add tooltips for better SEO
