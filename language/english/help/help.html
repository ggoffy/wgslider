Help: wgSlider

Description

The wgSlider module lets you create and manage image sliders in XOOPS.

Install/uninstall

No special measures necessary, follow the standard installation process and extract the wgslider folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.

Detailed instructions on installing modules are available in the XOOPS Operations Manual

Features

Create categories, upload images, and build slideshows for frontend display.
You can configure slideshow behavior and templates from the module administration area.
